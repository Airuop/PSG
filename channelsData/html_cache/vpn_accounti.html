GreatFireVPN
یک سرویس VPN (شبکه خصوصی مجازی) است که با تمرکز ویژه روی دور زدن فیلترینگ و محدودیت‌های شدید اینترنتی طراحی شده است؛ به‌خصوص برای کشورهایی با فایروال‌های سخت‌گیرانه مانند چین، روسیه، ایران، ترکمنستان و میانمار.
🚀 ویژگی‌های کلیدی
🔹 دور زدن فایروال‌های سنگین
طراحی‌شده برای محیط‌های با سانسور شدید اینترنت و استفاده از چندین روش مختلف برای عبور از محدودیت‌ها.
🔹 سازگاری با دستگاه‌های مختلف

امکان استفاده بدون ایمیل یا ساخت حساب کاربری، همراه با ۳۰ روز استفاده رایگان.
🔹 حریم خصوصی

🔹 Split Tunneling
امکان انتخاب اینکه کدام برنامه‌ها از VPN استفاده کنند و کدام بدون VPN متصل شوند.

آموزش ترکیب کانفیگ V2ray و سایفون در اندروید

🔹 مرحله ۱: تنظیم V2RayNG
وارد برنامه V2RayNG بشید و به بخش Settings برید.
در پایین صفحه، گزینه Mode رو از حالت VPN به Proxy Only تغییر بدید.

🔹 مرحله ۲: اضافه کردن کانفیگ
یک کانفیگ سالم (که مطمئن هستید حجم داره) وارد کنید، ولی فعلاً بهش وصل نشید.

🔹 مرحله ۳: تنظیمات Psiphon
وارد برنامه Psiphon بشید.
از بخش Options → VPN Settings:
گزینه Don't tunnel selected apps رو فعال کنید.
بعد از لیست، برنامه V2RayNG رو انتخاب کنید و روی Done بزنید.

🔹 مرحله ۴: تنظیم پراکسی در Psiphon
وارد بخش Proxy Settings بشید.
گزینه Use the following settings رو فعال کنید.
مقادیر رو اینطوری وارد کنید:
IP: 127.0.0.1
Port: 10808

🔹 مرحله ۵: اتصال
اول داخل V2RayNG روی Connect بزنید،
بعد Psiphon رو روشن کنید.
چند دقیقه صبر کنید تا اتصال کامل برقرار بشه.
